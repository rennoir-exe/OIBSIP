# Data Analytics Internship – Oasis Infobyte

This repository contains the projects completed as part of the Data Analytics Internship at Oasis Infobyte.

## Tasks Completed

### Task 1 – Exploratory Data Analysis (EDA)
- Project: McDonald's Nutrition Data Analysis
- Performed EDA to understand:
  - Calorie distribution
  - High-calorie food items
  - Relationship between sugar and calories
  - Nutritional insights using visualizations
- Files:  
  - `Task-1-EDA/McDonalds_Nutrition_EDA.ipynb`  
  - `Task-1-EDA/mcdonalds_nutrition.csv`  

### Task 2 – Customer Segmentation
- Project: Marketing Analytics – Customer Segmentation
- Segmented customers using K-Means clustering based on income and spending
- Files:  
  - `Task-2-Customer-Segmentation/Customer_Segmentation.ipynb`  
  - `Task-2-Customer-Segmentation/ifood_df.csv`
