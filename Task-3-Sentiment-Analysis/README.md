# Task 3: Twitter Sentiment Analysis

## Project Overview
This project focuses on performing sentiment analysis on Twitter data to understand
public opinion. Tweets are classified into Positive, Neutral, and Negative sentiments
using data analysis and visualization techniques.

## Dataset
- Source: Kaggle
- File Name: Twitter_Data.csv
- Columns:
  - clean_text: Cleaned tweet text
  - category: Sentiment label (-1 = Negative, 0 = Neutral, 1 = Positive)

## Objectives
- Load and explore Twitter sentiment dataset
- Clean missing values
- Analyze sentiment distribution
- Visualize sentiment categories

## Tools & Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib

## Steps Performed
1. Loaded the Twitter dataset using Pandas
2. Checked dataset structure and missing values
3. Removed null values to clean the data
4. Analyzed sentiment distribution using value counts
5. Visualized sentiment distribution using bar chart

## Results
The analysis shows that the dataset contains tweets with positive, neutral,
and negative sentiments. Positive tweets were the most frequent, followed by
neutral and negative tweets.

## Conclusion
This project helped in understanding how sentiment analysis can be performed
on real-world text data. Such analysis is useful for understanding public opinion
and customer feedback on social media platforms.
