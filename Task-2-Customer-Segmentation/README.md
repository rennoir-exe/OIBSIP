Task 2 – Customer Segmentation using K-Means Clustering

Objective:
The goal of this task is to segment customers based on their income and total spending behavior using K-Means clustering. This helps businesses understand customer groups and design targeted marketing strategies.

Dataset:
iFood Marketing Analytics Dataset

Files in this folder:
- Customer_Segmentation.ipynb
- ifood_df.csv

Steps Performed:
1. Loaded and explored the dataset
2. Selected relevant features (Income and Total Spending)
3. Scaled the data using StandardScaler
4. Used the Elbow Method to determine optimal clusters
5. Applied K-Means clustering
6. Visualized clusters and analyzed customer segments

Result:
Customers were successfully segmented into three distinct groups:
- Low income – low spending customers
- Medium income – medium spending customers
- High income – high spending customers

Conclusion:
Customer segmentation using K-Means provides valuable insights into customer behavior and can help businesses improve personalized marketing and decision-making.
