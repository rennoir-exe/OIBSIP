## Data Analytics Internship – Oasis Infobyte

### Task 1: Exploratory Data Analysis (EDA)

**Project:** McDonald's Nutrition Data Analysis

### Description:
This project performs Exploratory Data Analysis on McDonald's nutrition dataset to understand:
- Calorie distribution
- High-calorie food items
- Relationship between sugar and calories
- Nutritional insights using visualizations

### Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

### Files:
- `McDonalds_Nutrition_EDA.ipynb` – EDA notebook
- `mcdonalds_nutrition.csv` – Dataset
